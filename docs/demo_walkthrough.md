# Demo Walkthrough

1. Open the dashboard and review the model comparison page.
2. Submit a new applicant profile on the prediction page.
3. Receive an immediate prediction and note the `request_id`.
4. Navigate to explainability visualization and refresh until the explanation is ready.
5. Compare SHAP and LIME top features and review the ELI5 summary.
6. Review fairness metrics, monitoring metrics, and audit logs.
7. Submit feedback for the explanation quality.
